# 🌐 Streamlit Cloud 배포 가이드

## 📋 배포 준비

### 1. GitHub 저장소 생성

1. [GitHub](https://github.com)에 로그인
2. 우측 상단 `+` → `New repository` 클릭
3. Repository 이름 입력 (예: `influencer-contract-analyzer`)
4. Public으로 설정
5. `Create repository` 클릭

### 2. 코드 업로드

터미널에서 실행:

```bash
cd /Users/woobin/Documents/서댐/바이브코딩/vybemgmt

# Git 초기화 (이미 완료됨)
git init

# 파일 추가
git add .

# 커밋
git commit -m "Initial commit: 인플루언서 계약서 분석 시스템"

# GitHub 저장소 연결 (본인의 GitHub 주소로 변경)
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# 업로드
git branch -M main
git push -u origin main
```

## 🚀 Streamlit Cloud 배포

### 3. Streamlit Cloud 가입 및 배포

1. **Streamlit Cloud 접속**
   - https://streamlit.io/cloud 방문
   - `Sign up` 클릭
   - GitHub 계정으로 로그인

2. **앱 배포**
   - `New app` 버튼 클릭
   - Repository 선택: `YOUR_USERNAME/YOUR_REPO_NAME`
   - Main file path: `app.py`
   - App URL 설정 (원하는 주소)

3. **API 키 설정 (중요!)**
   - `Advanced settings` 클릭
   - `Secrets` 섹션에서 다음 내용 입력:

```toml
# Secrets 설정
OPENAI_API_KEY = "sk-proj-your-api-key-here"
LLM_MODEL = "gpt-4o"
API_TIMEOUT = "60"
MAX_RETRIES = "3"
USD_TO_KRW_RATE = "1300"
```

4. **배포 완료**
   - `Deploy!` 버튼 클릭
   - 약 2-3분 후 앱 실행 완료
   - 생성된 URL로 접속 가능 (예: `https://your-app.streamlit.app`)

## ⚠️ 주의사항

### 환경 변수 설정 필수!

- `.env` 파일은 GitHub에 업로드되지 않습니다 (.gitignore에 포함)
- Streamlit Cloud의 Secrets 섹션에서 직접 설정해야 합니다
- API 키가 없으면 앱이 작동하지 않습니다!

### 무료 플랜 제한

- **리소스**: 1GB RAM
- **동시 사용자**: 무제한이지만 성능 제한
- **앱 개수**: 무제한
- **배포**: 자동 (GitHub push 시)

### API 비용

- OpenAI API 사용료는 별도 발생
- 사용량에 따라 과금되므로 모니터링 필요
- 월 사용 한도 설정 권장

## 📊 배포 후 관리

### 로그 확인
- Streamlit Cloud 대시보드에서 실시간 로그 확인 가능
- 에러 발생 시 로그에서 원인 파악

### 앱 재시작
- Streamlit Cloud 대시보드 → `⋮` 메뉴 → `Reboot app`

### 앱 삭제
- Streamlit Cloud 대시보드 → `⋮` 메뉴 → `Delete app`
