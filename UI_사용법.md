# 🌐 웹 UI 사용 가이드

## 실행 방법

### 방법 1: 실행 스크립트 사용 (권장)
```bash
./run_ui.sh
```

### 방법 2: 직접 실행
```bash
python3 -m streamlit run app.py
```

또는

```bash
/usr/local/bin/python3 -m streamlit run app.py
```

## 접속 방법

실행하면 자동으로 브라우저가 열립니다.
- **로컬 주소**: http://localhost:8501
- **네트워크 주소**: http://192.168.x.x:8501 (다른 기기에서 접속)

## 사용 방법

### 1️⃣ API 키 설정 확인
- 좌측 사이드바에서 API 키 상태 확인
- ✅ 녹색: 정상 설정됨
- ❌ 빨간색: `.env` 파일에 API 키 설정 필요

### 2️⃣ PDF 업로드
1. "📄 계약서 업로드" 영역의 업로드 버튼 클릭
2. 분석할 계약서 PDF 선택
3. 파일 정보 자동 표시

### 3️⃣ 비용 확인
- 예상 토큰 수 표시
- USD/KRW 예상 비용 표시
- 사용할 AI 모델 확인

### 4️⃣ 분석 시작
1. "📊 계약서 분석 시작" 버튼 클릭
2. 진행률 바로 상태 확인
3. 분석 완료 대기 (수 분 소요)

### 5️⃣ 결과 확인
- **📄 분석 리포트 탭**: 분석 결과를 바로 확인
- **💾 다운로드 탭**: Markdown 파일로 다운로드

## 주요 기능

### 🔍 자동 분석
- AI 기반 계약서 자동 검토
- 위험 요소 자동 탐지
- 법적 근거 제시

### 📊 위험도 분류
- 🔴 **치명적 위험**: 즉시 대응 필요
- 🟡 **불리한 조항**: 협상 필수
- 🔵 **확인 필요**: 명확화 요구

### 🎯 협상 전략
- 구체적인 수정안 제공
- 역제안 문구 제시
- 협상 우선순위 제안

## 설정 옵션

### API 제공자 선택
좌측 사이드바에서 선택 가능:
- **자동 선택**: `.env` 파일 기반 자동 선택
- **OpenAI**: GPT-4 사용
- **Anthropic**: Claude 사용

## 출력 파일

분석이 완료되면 다음 위치에 저장됩니다:
```
./output/계약서분석리포트_[파일명]_[타임스탬프].md
```

## 문제 해결

### 포트가 이미 사용 중인 경우
```bash
# 다른 포트로 실행
streamlit run app.py --server.port 8502
```

### 브라우저가 자동으로 열리지 않는 경우
수동으로 접속:
```
http://localhost:8501
```

### API 키 오류
1. `.env` 파일 확인
2. `OPENAI_API_KEY` 또는 `ANTHROPIC_API_KEY` 설정 확인
3. 앱 재시작

## ⚠️ 주의사항

1. **법적 자문 대체 불가**
   - AI 자동 분석 결과입니다
   - 중요 계약은 변호사 검토 필수

2. **API 비용**
   - 분석마다 API 사용료가 발생합니다
   - 비용 추정 후 진행하세요

3. **개인정보 보호**
   - 민감한 정보 포함 시 주의
   - 업로드된 파일은 임시 저장 후 삭제됩니다
